| 数据仪表盘 | dashboard, technical, levels |
| 艺术摄影 | minimal, artistic, textured |

| 情绪 | 推荐风格 |

## 使用示例

```bash
# 拉取glassmorphism风格
typeui pull glassmorphism

# 拉取elegant风格
typeui pull elegant

# 查看所有风格
typeui list

# 应用到现有HTML
python scripts/apply-style.py --style glassmorphism --input slides.html
```
