# 幻灯片模板库

## 1. 封面页模板

### 1.1 居中标题式
```
┌────────────────────────────────────────┐
│                                        │
│          🎯 主标题（大字）              │
│          ──────────────                │
│          副标题/导读（小字）            │
│                                        │
│          作者名 | 日期                  │
│                                        │
└────────────────────────────────────────┘
```

**适用场景**：正式报告、学术演讲

## 每页内容规则

| 元素 | 最大数量 | 建议数量 |
|-----|---------|---------|
| 文字行数 | 8行 | 4-6行 |
| 要点数量 | 6个 | 3-5个 |
| 字体大小（标题） | 48px | 36-42px |
| 字体大小（正文） | 24px | 18-20px |
| 颜色种类 | 4种 | 2-3种 |

## 动画效果建议

| 元素 | 入场动画 | 延迟 |
|-----|---------|------|
| 标题 | 淡入上滑 | 0ms |
| 副标题 | 淡入 | 200ms |
| 要点列表 | 逐项滑入 | 每条+150ms |
| 图片 | 缩放淡入 | 300ms |
| 数据 | 数字递增 | 500ms |
